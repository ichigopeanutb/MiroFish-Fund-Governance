# LP-Facing Material Pack

This folder contains private-beta materials for introducing MiroFish Fund
Governance Edition to LPs, fund managers, IC / LPAC reviewers, and technical
trial users.

These materials are designed for controlled conversations. They should not be
used as public fundraising materials, legal advice, tax advice, accounting
advice, investment advice, or a representation of any real fund performance.

## How To Use The Pack

For a first nontechnical fund-manager reviewer, send only the Chinese PDF and a
short message asking for three reactions. Use the first-batch operations pack to
track who you invited, the demo script only when you schedule a walkthrough, and
the feedback form after the reviewer has responded.

## Positioning Boundary

MiroFish Fund Governance Edition is a decision-rehearsal and governance
simulation tool. It helps reviewers inspect whether fund terms, capital calls,
portfolio events, governance decisions, evidence trails, and meeting packs are
represented in a useful way.

It is not a fund administrator, investment advisor, legal advisor, accounting
system, investor portal, or performance prediction product.

## Recommended Trial Flow

1. Send the one-page overview.
2. Schedule a 30-minute walkthrough.
3. Run the synthetic demo only.
4. Ask the reviewer to focus on one role:
   - LP reviewer
   - Fund manager / GP reviewer
   - IC / LPAC reviewer
   - Technical operator
5. Capture feedback using the trial feedback form.
6. Translate feedback into the next product milestone.

## Data Boundary

Do not enter real LP confidential data, real fund documents, side letters,
investor identifiers, bank information, tax records, legal advice, or accounting
records into the demo. Use synthetic data until a formal data handling process
exists.
