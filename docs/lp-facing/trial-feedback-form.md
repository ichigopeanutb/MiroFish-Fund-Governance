## Role-Specific Questions

### For LP Reviewers

- Would this help you understand fund process, capital status, and governance
  readiness?
- What would you need to see before trusting this as a discussion aid?
- Which LP-facing output matters most: report, meeting pack, evidence appendix,
  capital summary, or scenario comparison?

### For Fund Managers / GPs

- Would this reduce preparation work for LP / IC / LPAC conversations?
- Which part of fund operations is still unrealistic or missing?
- Would you use this before a real meeting, after a meeting, or only as an
  internal planning tool?

### For IC / LPAC Reviewers

- Are decision points framed at the right level?
- Is the packet too detailed, too thin, or close to useful?
- Which approval / waiver / remediation path should be added next?

### For Technical Operators

- Was setup clear?
- Did the access-code flow make sense?
- Which output files are most useful for integration or downstream workflows?
