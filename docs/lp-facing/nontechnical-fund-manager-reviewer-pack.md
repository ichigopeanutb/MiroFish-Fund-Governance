# Nontechnical Fund Manager Reviewer Pack

This is the short packet to send to a nontechnical fund manager, senior advisor,
or LP-facing reviewer. It is designed for review without GitHub access.

## What To Send

1. This reviewer pack.
2. One-page overview.
3. Demo page screenshot.
4. Sample completed report.
5. Three feedback questions.

Do not send the private GitHub repo unless the reviewer specifically wants to
inspect the technical setup.

## One-Sentence Explanation

MiroFish Fund Governance Edition is a synthetic fund-governance simulation tool
that helps rehearse capital calls, fund terms, waterfall, IC / LPAC decisions,
LP communication, evidence trails, and meeting-pack preparation.

## Three Questions For The Reviewer

1. If you were a fund manager or LP-facing advisor, which part of this output
   would be most useful in a real conversation?
2. Which part is unclear, too technical, or not yet credible?
3. What would need to be added before you would introduce this to another fund
   manager, LP, IC member, or LPAC reviewer?

## Suggested Message

Hi [Name],

I am testing an early version of MiroFish Fund Governance Edition. It uses
synthetic data to rehearse fund operations, LP communication, governance review,
capital calls, waterfall, and meeting-pack preparation.

You do not need GitHub or any technical setup. I only want your judgment on
whether the screen, sample report, and product direction make sense from a fund
manager / LP-facing perspective.

Could you look at the attached packet and answer three questions?

1. What is immediately useful?
2. What is unclear or not credible yet?
3. What would make this worth a second conversation?

All materials are synthetic and do not include real LP or fund data.
